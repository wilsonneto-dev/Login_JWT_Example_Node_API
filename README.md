## Login JWT on a Node API
#### Simple example of a JWT authentication on a Node API.

- Encript user's password in registration
- Generate a JWT token on login route
- Protect routes verifying JWT token

Using libs:

- jsonwebtoken
- bcrypt

- dotenv
- Mongoose
- Express
- Cors
- Nodemon
